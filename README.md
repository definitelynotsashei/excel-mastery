# Project Starter Pack

This folder is a reusable bootstrap pack for a new software project.

It includes:
- source-of-truth project docs
- a reusable Codex workflow prompt file
- a minimal folder structure for app, feature, shared, style, test, and docs work

## Suggested Use
1. Copy this entire folder into a new project workspace.
2. Rename the folder to match the new project.
3. Replace the bracketed placeholders in the markdown files.
4. Adjust verification commands in `PROJECT_STATE.md`.
5. Start the first Codex session using the startup prompt in `AGENTS.md`.
